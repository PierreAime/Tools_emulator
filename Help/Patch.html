Patch to ROM Window
Back to Contents

The Patch to ROM Window allows you to patch .fzep files to a valid ROM file for use in other applications. The following features are provided:

Input ROM
This button selects a *.*64 file and will accept it if it is determined to be a valid F-Zero X ROM. The Output ROM (see below) field will be changed automatically.

Output ROM
The filename of the new ROM to be created when patched. Cannot be the same as Input ROM. This is done to prevent overwriting of source ROM images.

Selected ROM:
Displays the detected version of the selected ROM file, if any.

Selected Course
Specifies the target track location in ROM with a drop-down list and whether to replace it with course data loaded from a .fzep file.

Recalculate Header Checksum
Required if the patched ROM is intended to be used immediately for gameplay. Recalculates the N64 header checksums needed for the game to boot. The file rn64crc.exe is required for this function to work.

Patch
Performs all actions selected by other controls in the window.
