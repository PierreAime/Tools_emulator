F-Zero Execution Project: F-Zero X

F-Zero Execution Project is a ROM-hacking level editor for F-Zero X. Using an interface designed to provide only the desired functions at any given time, a configurable work environment is provided to better enable you to create courses for the game.

The program is designed to be accessible to those who have backed up ROM images of the game as well as those who have the F-Zero X Expansion Kit for Nintendo 64DD. This program requires no ROM image to function and is relatively lightweight. Features are supplied to make interchange of courses between users as easy as possible.

When you first launch the program, only a menu bar will be provided to you. You will need to activate windows manually for content to be displayed on-screen. Internally, all editing is done behind-the-scenes, so no window is required to be present at any given time. This gives you more control over how your editing environment is configured.
